# Quick Start Guide

## Prerequisites
- Node.js installed

## 1. Start Backend
Open a terminal in the `Backend` folder:
```powershell
cd Backend
npm start
```

## 2. Start Frontend
Open a new terminal in the `Frontend` folder:
```powershell
cd Frontend
npm run dev
# OR for production mode
npm start
```
*Expected Output:* `Ready in ... ms` on `http://localhost:3000`

## 3. Access the App
- Open **http://localhost:3000**
- Login: `admin@hospital.com` / `admin123`
